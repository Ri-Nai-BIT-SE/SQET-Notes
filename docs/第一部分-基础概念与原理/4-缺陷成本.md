# 缺陷成本（Cost of Defects）

**核心规律**：修复缺陷的成本随着发现时间的推迟呈**指数级增长**。

**成本增长规律**：
- **需求阶段发现**：成本最低
- **设计阶段发现**：成本增加
- **编码阶段发现**：成本进一步增加
- **测试阶段发现**：成本显著增加
- **发布后发现**：成本最高（可能达到需求阶段的100倍以上）

**【录音重点】**：修复成本随着生命周期推移（从需求到发布），呈**几何级数**增长。

**启示**：
- 强调早期发现缺陷的重要性
- 通过评审、审查等静态方法在早期发现缺陷
- 将质量工作前移，从昂贵的后期返工转化为前期的缺陷发现
