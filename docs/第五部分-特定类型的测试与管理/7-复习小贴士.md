# 💡 复习小贴士

### 关于填空/选择

*   如果问到"达到路径覆盖需要多少用例"，如果没有循环，路径数是固定的；如果有循环，通常是**无法达到100%路径覆盖**的（选"无法达到"或类似的选项）。

### 关于判断题

*   "测试是为了证明程序没有错误" -> **错**（是为了发现错误）。
*   "自动化测试能完全取代人工测试" -> **错**。
*   "测试越早进行越好" -> **对**（W模型思想）。

### 关于大题

*   如果考**黑盒**，记得画**等价类表**。
*   如果考**白盒**，记得画**控制流图**和算**V(G)**。
*   如果考**案例**，看着图表里的曲线，那一根线飙高了就说哪里有瓶颈，多写几个专业术语（响应时间、吞吐量、资源利用率）。
