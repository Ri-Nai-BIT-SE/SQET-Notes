# 兼容性与本地化测试

### 🌍 兼容性测试 (Compatibility Testing)

*   **兼容性测试**：不仅测软件，还要测**硬件**（打印机、显卡）、**网络**（弱网）、**数据**（新旧版本数据迁移）。

### 兼容性矩阵

测试不同组合的兼容性：

- **硬件兼容性**：不同硬件配置
- **软件兼容性**：不同软件版本
- **操作系统兼容性**：Windows、Linux、macOS 等
- **浏览器兼容性**：Chrome、Firefox、Safari、Edge 等
- **网络兼容性**：不同网络环境（弱网、断网等）
- **数据兼容性**：新旧版本数据迁移
