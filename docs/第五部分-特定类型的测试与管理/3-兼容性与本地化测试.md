# 兼容性与本地化测试

### 🌍 兼容性测试 (Compatibility Testing)

**定义**：验证软件与其依赖环境（硬件、平台、软件）的依赖程度。

#### 硬件兼容性

*   **整机**：不同品牌、型号的计算机
*   **板卡**：显卡、声卡、网卡等
*   **外设**：打印机、扫描仪等

#### 软件兼容性

*   **操作系统**：Windows、Linux、macOS 等
*   **数据库**：MySQL、Oracle等不同版本
*   **浏览器**：Chrome、Firefox、Edge、Safari
  - *重点*：**浏览器兼容性矩阵**（曾考过设计题）
*   **中间件**：不同版本的中间件

#### 数据兼容性

*   **不同版本间的数据迁移**：新旧系统迁移
*   **数据格式兼容性**：不同格式的数据导入导出

#### 解决策略

*   租用/借用设备
*   使用第三方测试实验室

### 兼容性矩阵

测试不同组合的兼容性：

- **硬件兼容性**：不同硬件配置
- **软件兼容性**：不同软件版本
- **操作系统兼容性**：Windows、Linux、macOS 等
- **浏览器兼容性**：Chrome、Firefox、Safari、Edge 等（**考点**：浏览器兼容性矩阵设计题）
- **网络兼容性**：不同网络环境（弱网、断网等）
- **数据兼容性**：新旧版本数据迁移
