# 🚀 综合案例演示（模拟大题）

**题目**：
```c
void test(int A, int B, int X) {
    if (A > 1 && B == 0)  // 判定节点 1
        X = X / A;        // 语句节点 2
    if (A == 2 || X > 1)  // 判定节点 3
        X = X + 1;        // 语句节点 4
    printf("%d", X);      // 结束节点 5
}
```
**问题：请画出控制流图，计算圈复杂度，并找出独立路径。**

### 解题思路：

1.  **拆分复合条件（画图关键）**：
    *   第一个 `if (A > 1 && B == 0)` 要拆成：先判 `A>1`，如果是真，再判 `B==0`。
    *   第二个 `if (A == 2 || X > 1)` 要拆成：先判 `A==2`，如果是真直接进；如果是假，再判 `X>1`。

2.  **画图 (脑补一下图形)**：
    *   节点①: `A > 1` ? (F去③, T去②)
    *   节点②: `B == 0` ? (F去③, T去语句块)
    *   ...以此类推。

3.  **计算圈复杂度 V(G)**：
    *   假设我们拆分后有 4 个判定节点（因为有两个复合条件）。
    *   公式：`V(G) = 判定节点数 + 1` = 4 + 1 = 5。
    *   也就是说，你需要找出 5 条独立路径。

4.  **列出路径**：
    *   Path 1: 全 False。
    *   Path 2: 第一个 if 全 True，第二个 if 全 False。
    *   ... (以此类推，凑够5条)。
