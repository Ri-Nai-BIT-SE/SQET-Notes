## 📝 使用建议

1.  **闭卷作答**：请尽量在不看书的情况下完成，模拟真实考试环境（限时120分钟）。
2.  **大题必练**：特别是**白盒测试的圈复杂度计算**和**黑盒测试的等价类划分**，一定要亲手画图、填表。
3.  **案例分析**：学会使用专业术语（如"响应时间"、"吞吐量"、"内存泄漏"）来回答问题，不要用大白话。
4.  **查漏补缺**：做完后对照参考答案，发现错题立即回去翻看对应的笔记章节。
