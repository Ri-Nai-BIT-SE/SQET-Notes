# 4. 考前30分钟"抢分"清单

进考场前，最后看一眼这几个东西：

1.  **三个圈复杂度公式**：
    *   `V(G) = 区域数`
    *   `V(G) = E - N + 2`
    *   `V(G) = P + 1`
2.  **逻辑覆盖强弱顺序**：
    *   语句 < 判定 < 条件 < 判定/条件 < 条件组合 < 路径
3.  **黑盒方法的适用场景**：
    *   **无穷输入** -> 等价类
    *   **规定范围/个数** -> 边界值
    *   **条件组合/逻辑约束** -> 因果图/判定表
4.  **性能指标阈值**（案例题用）：
    *   CPU > 90~95% -> 瓶颈
    *   内存只升不降 -> 泄漏
5.  **核心定义**：
    *   **验证** (过程) vs **确认** (结果)
    *   **驱动** (向上) vs **桩** (向下)
